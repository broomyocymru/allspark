# Virtual Private Cloud (VPC)

Core building block of an AllSpark project, this configures the core networking, storage and security. By default initiating an AllSpark project will create a default management VPC, which includes a bastion server.


## Usage
```bash
allspark sparks add vpc --name "dev"
```
